
\begin{oldtt}
We will make extensive use of the fact that orderings compose (see lemma \lref{orderingsCompose}) and we will use it in the following form:
\begin{observation}
\llabel{orderTightening}
If \catcw is a range category and
\[
\begin{array}{c p{0.5cm} c }
\nudgeup{1.0cm}
\bOb{x} &         & \bOb{y}\\[0.3cm]
\end{array}
\begin{arrows}
\drawMorphismArcing{x}{y}{a}{f}{20}{20}[0.5]
\drawMorphismArcing{x}{y}{b}{z}{-20}{-20}[0.5]
\ncarrrecursive{x}{x}{170}{80}{10}\alabel{\rst{i}}
\ncarrrecursive{y}{y}{80}{170}{10}\alabel{\rng{j}}
\end{arrows} 
\] 
in \catcw then if $\rst{z} \leq \rst{i}$,
$z \leq f$ and $\rng{z} \leq \rng{j}$ then
$z \leq \rst{i} \circ f \circ \rng{j}$.
\end{observation}
\end{oldtt}
