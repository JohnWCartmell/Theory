
\subsection{Simplify and follow Cockett more closely}
If \catcw is a range category, define a functor $K_a: \catc \morph \Par$ by defining for each object $x$ of \catc, $K_a(x)=Hom_{\Stk}(a,x)$ and by defining $K_a(j)$, for $j:y_1 \morph y_2$ in \catc, to be the partial function  defined at a zigzag 
$F=\zigzagTuple{f}{n}\in Hom_{\Stk}(a,x)$ by
\begin{align*}
K_a(j)(F) &= F :: \tuple{j}  &&\mbox{ if $\overline{f_n \circ j} = \overline{f_n}$,} \\
             &=\bot        && \mbox{otherwise.}
\end{align*}

Note that the condition $\overline{f_n \circ j} = \overline{f_n}$ ensures
that $F :: \tuple{j}$ is a tightg zigzag and therefore that
$F :: \tuple{j}= F \circ \tuple{j}$.

$K_a$ is a range functor by arguments made by Cockett.

The RR.5 condition is required to ensure that every tight
zigzag has a unique shortening. 




\subsection{Requirements for $K_a$}

From memory require:
\begin{enumerate}[(i)]
\item If $f: a \morph x$ in \catcw then 
\[
K_a(f)(\tuple{\bar{f}})=\tuple{f}
\]
\item If $f: a \morph x$ and $g: a \morph x$ in \catcw then 
\[
K_a(f)(\tuple{\bar{f}\circ\bar{g}})=\tuple{\bar{g} \circ f}
\]
and, therefore,
\[
K_a(g)(\tuple{\bar{f}\circ\bar{g}})=\tuple{\bar{f} \circ g}
\]
\end{enumerate}
